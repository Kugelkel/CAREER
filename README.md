### ☑️ 프로젝트

#### (1) 대동 BI 프로젝트

**기간** : 2024년 10월 ~ 2024년 12월 (약 3개월)

**성과** : DATALAKE 데이터를 BI 마트로 정제 및 설계하고 대시보드 개발과 교육을 통해 데이터 품질 향상, 자동화 구현, 운영 효율 개선, 사용자 적응도 증가를 모두 달성하여 고객사의 Tableau 도입 초기 성공 사례로 자리매김.

- 10개 이상의 마트 개발.
- 13개의 대시보드 개발.
- 운영 효율 80% 이상 개선.
- 사용자 요구 만족도 95% 이상.

**역할**

- **데이터 소스 정제 및 연동 :**
  스노우 플레이크에 적재된 MES, 엑셀 형태의 데이터 소스를 기반으로 생산 부문 지표에 맞춘 Tableau 대시보드 전용 마트 10개 이상 개발 및 연동.
  윤달 데이터로 인한 오류를 파악하여, 해당 문자열 처리 구문을 날짜 형태의 구문으로 수정하여 적용
- **금융 상품 대시보드 시각화 :**
  금융 상품 별 분석 요건을 수집하여 맞춤형 대시보드 구조 설계.
  기간별 성과 비교, 시장 트렌드 파악을 위한 대시보드 개발.
  사용자 요구에 따라 필터, 드릴다운, 인터렉티브 기능 구현.
- **맞춤형 컨설팅 제공 :**
  고객사가 대시보드를 효과적으로 활용할 수 있도록 가이드라인 문서 작성.
  기간별 대시보드 구성방안, 데이터 해석 방법, 사용자 교육 등 컨설팅 제공 및 가이드라인 문서 작성.
